Installation
------------
Run:
	make

Running make will generate a separate build directory that
contains all binaries for llvm and the projects.

Packages
--------
   o llvm          - llvm source code
   o projects      - llvm passes for different projects

Other
-------
   o setup.sh      - for build system (ignore)
   o package.sh    - for build system (ignore)
